> Extend [`hyper-media-control`](https://github.com/OrionNebula/hyper-media-control) with support for [`Google Cast`](https://developers.google.com/cast/) devices.

## Installation

Add `hyper-media-control` and `hyper-media-control-cast` to your Hyper configuration.

## Configuration

`hyper-media-control-cast` doesn't define any configuration options.

## Notes

This plugin is in the early development stages, and not super reliable.
